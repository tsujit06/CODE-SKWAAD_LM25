# Amazon-Clone-01
A frontend clone of E-commerce website of Amazon. This is created using HTML,CSS and Vanilla Javascript.Used to demonstrate chat bot using NLP, The page is created for big screens i.e, laptops, PCs only.

# Technologies:
* HTML5
* CSS3
* Javascript
* Netlify
* Python 3
* Flask
* NLP
* pyttsx3
* speechRecognizer
